List of tools I've built and find useful, in no particular order:


### Ollama openrouter proxy
[ollama-openrouter-proxy](https://github.com/aljazceru/ollama-openrouter-proxy) -  Ollama proxy that uses free models available on openrouter.

### Timeconvert.org
[timeconvert.org](https://timeconvert.org/) - Convert timestamps and dates into various formats, including Unix, ISO 8601, UTC, and more.

### RSS / Atom debugger
[debug-rss](https://aljazceru.github.io/debug-rss/) - Your RSS Army Knife - Analyze and debug any RSS or Atom feed

### HFM - Hetzner Firewall Manager
[HFM](https://github.com/aljazceru/hfm) - A Python tool to manage Hetzner Robot Firewall configurations via the API.

### ROCM Monitoring
[rocm-monitoring](https://github.com/aljazceru/rocm-monitoring) - A set of scripts and tools to monitor AMD GPUs using ROCm stack.

### Azure firewall updater  
[azure-firewall-updater](https://github.com/aljazceru/azure-firewall-updater) -  Add my current ip to azure firewalls allowed ip list
